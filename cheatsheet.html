HTML Structure reccomendations

Header

The header contains the title, a link to our css style sheet, and an H1 element with the title of the page with some css styling and images to state the branding.

Nav

The Nav is the navigation of the website, here you will put links to other pages or redirect to lower parts if this page.

Section

contained in the body element, you use section to group content that is related to eachother.

Article

This element is used to specify an independent, self contained content. It needs to make sense on its own.

Headers

All headers need to be an introduction to the content below it. Do not put an h1 header below an h2 header. this causes a lot of issues with SEO and other features required to aid in the disableds use of a website.

Aside

The aside element is used when you have content that is related to a section but is more supplimental then required. Like this image of our structure next to the content you are currently reading.

Footer

Like this header, this element contains information about the website that is related to its contents. like copyright and links to related content. Footers can be used within sections to help with further organizing your structure.

HTML Generic Elements

These elements are headings for content. The order is very important. Do not use h1 after h2.

This element is the paragraph element. Used to show text in a direct manner.

HTML Attribute tags

These are all the common attribute tags an element tag might use.

href

This is usually used by the link,a,and nav elements to find the source or link to which it is reffering or redirecting to.

src

This is used by the img, and video tags to find the path where the desired content is located.

width & height

typically used by the img and video tags to know what size to display the content in. It is best practice to always use this.

alt

Typically used by the img tag, used to give an alternate text for the image. Good for SEO and some features used by those with disabilities.

style

used to add css styles to elements directly in line. do not use, but be aware of its usage.

lang

declares the language of the webpage. used at the top of the file.

title

this defines some extra information about an element.

class

used to group elements into a class so that the css files can style them on a larger scale for reusability.

id

Set this attribute to be used by other code within the file or in the styling sheet we have connected to this html file.

HTML Best Practices

These are the best practices to follow when starting a new project.

Use proper document structure

Always start with (!DOCTYPE html) at the top of the file to ensure that it renderes properly.

Always close tags

to avoid encountering validation errors always remember to have a closer tag for every tag you create.

Don't use inline styles

Keeping your style seperate makes it easier to update and change your content. as well as increase the readability.

Use alt attribute with images

it is not required, but it is very important to have a meaningful alt attribute for validation and accessibility reason. it provides context to screen readers, so it should be descriptive as to what the image contains.

Validate frequently

ensure that your file is valid consistently, to avoid missing errors early on. especially if the document is lengthy.

Place styling sheets within the head tag

This will allow the page to load faster.

Use meaningful tags

This improves readability and allows you to update and fix things a lot easier down the road.

Use lowercase markup

It is easier to read and maintain.

reduce the number of elements on a page

redability becomes tedious if you don't find a way to optomize the amount of tags being used.

background color

This is the background behind the text

Alignment

text-align:left;

this sets the horizontle alignment of the text, justify makes sure the left and right are parallel.

transform

text-transform: uppercase;

This allows you to change the case of the letters. includes capitalize.

indent

text-indent:50px;

allows you to indent the first line of the text.

letter spacing

letter-spacing: 3px;

This specifies how much space you want between each letter.

Line Height

line-height:0.8;

used to specify the space between lines.

word spacing

word-spacing:10px;

used to specify how much space you want between each word.

white space

white-space: nowrap;

specifies how white space inside an element is handled.

text shadow

text-shadow: 2px 2px 5px color

This element will put a shadow on the text.

The first number is the x offset, the second the y offset, the third is a blurr effect, and the last is the color name being set.
